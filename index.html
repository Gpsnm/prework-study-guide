HTML

  • The head element contains information about a webpage.
  • The body element represents the visible content shown to the user.
  • HTML elements include Text element, Image element and Header element
  • Elements used to divide and describe include meta related element, content division, semantic elements.
  • The title element defines the title for the webpage and is important for SEO

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicated how much space we want around the content inside an element.
  • There are 3 ways to link CSS inline-css, internal css and external css
  • a class selector can be used on multiple elements
  • The border property assigns a border you can also assign width and color. border: solid 5px gray
  • assigning colors in css can be done multiple ways including naming the color or using hexadecimal code or hex #

GIT

  • Git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: to switch back to the main branch
  • git pull origin main: to get the latest version of the main branch
  • git merge [branch]: merge the specified branch history into the current one
  • git commit -m "message": commit your staged content as a new commit
